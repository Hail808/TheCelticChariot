UPLOAD IMAGE FOR NEW LISTING
This is a test to test the upload image feature
The admin account is logged into and navigated to the admin catalogue. A new listing is created
and a placeholder image is uploaded. It then checks if the listing was successfully created
and deletes the listing.

Instructions:
    This test is verified to work on:
        -Google Chrome VER 141.0.7390.123
        -Python Ver 3.13
        -Selenium 4.38.0

    Before running test, make sure dev server is not already running

    To run test, in command line:
	    ./test-start.bat
	Or double click on test-start.bat
    This script will start the test program and start up the server and webhook

    To close the program, hit ENTER and check consoule for output. Also make sure to close the dev server
